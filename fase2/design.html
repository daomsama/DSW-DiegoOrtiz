Proposito del Sitio

Este sitio tiene como proposito ser una herramienta de aprendizaje que le permita a los usuarios de una manera didactica aprender todo lo relaciónado con etiquetas para el desarrollo de paginas con HTML

Etiquetas Semanticas

Según la biblia de HTML de Francisco Charte Ojeda Este tipo de etiquetas del lenguaje HTML son aquellas que dan un significado a las partes del documento estas son.

  • header: la cabecera de una web.
  • nav: establece un apartado con enlaces de la página, por ejemplo, un menú.
  • main: expresa el contenido principal de un documento.
  • section: define una sección de un documento.
  • article: indica que el contenido encerrado dentro de las etiquetas es un artículo.
  • aside: el contenido lateral de una página.
  • footer: hace referencia al pie de una página o sección.

    • CCS GRID

    Según la biblia de HTML de Francisco Charte Ojeda CSS es un lenguaje que trabaja junto con HTML para proveer estilos visuales a los elementos del documento, como tamaño, color, fondo,bordes, etc, y grid que traducido al español significa cuadricula, son tablas que permiten organizar elemntos en tablas en el sitio, ambos conceptos indican los estilos que se le pueden dar a las diferentes cuadriculas que puedan tener en un sitio.

    Una de las técnicas más simples para incorporar estilos CSS a un documento HTML es la de asignar los estilos dentro de las etiquetas por medio del atributo style. Por ejemplo el valor font-size: 20px. Este estilo cambia el tamaño por defecto del texto dentro del elemento a un nuevo tamaño de 20 pixeles.
